## Hey there! 👋

I'm Maria Eduarda, an undergraduate student in Mathematics at IMPA Tech, Brazil.

I enjoy building software that combines mathematics, education, and technology.

If an idea catches my attention, chances are I'll turn it into a project and keep improving it.

If you'd like to explore my work, feel free to check out my repositories!
